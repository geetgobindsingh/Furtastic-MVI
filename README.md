# Furtastic-MVI

This repo is a playground about best practices, using updated libraries and solutions in the Android world!

## ⚙️ Architecture

The main architecture of code based on MVI + CLEAN architecture.

## 🛠 Technologies

- Jetpack Compose
- CLEAN architecture
- MVI architectural pattern
- Coroutine Flow
- SqlDelight database
- Koin
- Navigation
- Ktor
- Version catalog
